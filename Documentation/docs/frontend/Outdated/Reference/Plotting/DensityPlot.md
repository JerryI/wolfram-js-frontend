```mathematica
DensityPlot[f, {x, xmin, xmax}, {y, ymin, ymax}]
```
makes a density plot of `f` as a function of `x` and `y`

:::warning
Only a flat-shading is supported for now
:::


```mathematica
DensityPlot[x y, {x,0,1}, {y,0,1}]
```
